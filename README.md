## Configuration

[`secrets.json`][secrets]:

```json
{
  "CVUT:ClientId": "test",

  "DATABASE_URL": "Host=localhost;Database=HonzaBotner;Username=honza-bot;Password=postgres",
  "Discord:Token": "test"
}
```
